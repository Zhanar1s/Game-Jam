# GONE game project
### Installation
1. Clone the github repo at:
```
git clone https://github.com/Zhanar1s/Game-Jam
```
2. Install the superlegendboy.ttf font as a system font
3. Install pygame if not installed
```
pip install pygame
```
4. Run the main.py file
```
python main.py
python3 main.py 
```
